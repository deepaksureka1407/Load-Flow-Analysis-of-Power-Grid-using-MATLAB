# Load Flow Analysis using Newton-Raphson Method

## Project Overview

This project simulates the load flow analysis of a 5-bus power system using the **Newton-Raphson method** in MATLAB. It calculates bus voltages, active/reactive power, and line losses iteratively.

## Features

- Newton-Raphson method for solving power system load flow.
- Built-in bus and line data for a 5-bus system.
- Iterative solution to determine voltage magnitudes and angles.
- Plots voltage profiles for each bus.
  
## Prerequisites

- MATLAB (any version supporting basic matrix operations)
  
## How to Run

1. Clone the repository or download the `loadflow_analysis.m` file.
2. Open MATLAB and navigate to the directory containing the `.m` file.
3. Run the script `loadflow_analysis.m` in MATLAB.

## Results

- The program outputs the voltage profiles (magnitude and angle) at each bus in the system.
- It also checks for convergence and displays the number of iterations.
